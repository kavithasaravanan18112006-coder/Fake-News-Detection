# 📰 Fake News Detection System

## 📌 Project Overview

The Fake News Detection System is a machine learning web application that detects whether a news article is REAL or FAKE using Natural Language Processing (NLP) techniques.

The project analyzes the text content of news articles and predicts authenticity using a trained Logistic Regression model with TF-IDF vectorization.

---

## 🚀 Features

* Detects fake and real news articles
* NLP-based text processing
* TF-IDF vectorization
* Logistic Regression classification
* Interactive Streamlit web interface
* Simple and user-friendly UI

---

## 🛠 Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit
* NLP (Natural Language Processing)

---

## 📂 Dataset

Dataset used:

* Fake.csv
* True.csv

Source:
https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

---

## ⚙️ Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Text Vectorization using TF-IDF
4. Train-Test Split
5. Logistic Regression Model Training
6. Prediction and Classification
7. Streamlit Web App Deployment

---

## ▶️ How to Run the Project

### Install Required Libraries

```bash
pip install pandas scikit-learn streamlit
```

### Run the Application

```bash
python -m streamlit run app.py
```
